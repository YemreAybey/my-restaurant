# my-restaurant
A simple restaurant page with home, menu and contact tabs.
Every page is created with JS.

# Used Stacks

- Javascript
- Html
- CSS
- Webpack
- NPM

## Getting started


Now open the html file via your browser. Click to navigation buttons.
